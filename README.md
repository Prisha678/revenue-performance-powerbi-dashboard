# Revenue Performance & Operations Analytics Dashboard
## Project Overview
This project presents an interactive Power BI dashboard built to analyze multi-year sales performance across regions and product categories. The objective was to monitor revenue trends, identify regional concentration risk, and evaluate operational efficiency using shipping metrics.
## Key Metrics
Total Sales: $2.26M; Total Orders: 4,922; Average Order Value: $459; Average Shipping Days: 3.96; Month-over-Month Growth: 4%
## Analytical Insights
West region contributes ~31% of total revenue, indicating regional dependency risk. Technology category leads overall sales performance. Revenue exhibits Q4 seasonal spikes, suggesting holiday-driven demand. Shipping performance remains within operational efficiency targets.
## Tools & Skills Used
Power BI; DAX (MoM Growth, Dynamic Top Region Detection); KPI Design; Time-Series Trend Analysis; Business Insight Communication
## Files Included
Power BI Dashboard (.pbix); Cleaned Sales Dataset (.csv); Dashboard Screenshot
